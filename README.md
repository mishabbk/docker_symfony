# Symfony 5 With Docker

### Start:
1. Run `make build`

2. Run `make start`

3. Run `make migrate`

### Stop
1. Run `make stop`

### Move docker to another project
**Need copy folder:**

- docker

**Need cope files:**

- .dockerignore

- env.prod

- docker-compose.yml

- docker-compose-prod.yml

- Dockerfile

- Makefile
